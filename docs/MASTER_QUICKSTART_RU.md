# Как подключить своего бота к Clientika

Простая инструкция для мастеров. Займёт 2 минуты.

---

## Шаг 1: Создайте бота в Telegram

1. Откройте Telegram и найдите **@BotFather** (это официальный бот Telegram для создания ботов)

2. Нажмите **Start** (или напишите `/start`)

3. Отправьте команду:
   ```
   /newbot
   ```

4. BotFather спросит: **«Как назвать бота?»**
   — Напишите название вашего бота (это то, что видят клиенты). Например:
   ```
   Маникюр Анна
   ```

5. BotFather спросит: **«Придумайте username»**
   — Это техническое имя, должно заканчиваться на `bot`. Например:
   ```
   ManikurAnna_bot
   ```
   Если имя занято — попробуйте другое (добавьте цифры или измените).

6. BotFather пришлёт сообщение с **токеном** — это длинная строка вида:
   ```
   8910981285:AAEr4tnCMl6jCO6z83vR9MaycaBhTMlNWL0
   ```
   **Скопируйте этот токен** (нажмите на него — он выделен как код).

---

## Шаг 2: Подключите бота к Clientika

1. Откройте **@Clientikabot** в Telegram (это главный бот платформы)

2. Если вы ещё не зарегистрированы — нажмите `/start`

3. Отправьте команду с вашим токеном:
   ```
   /addbot ВАШ_ТОКЕН
   ```
   Например:
   ```
   /addbot 8910981285:AAEr4tnCMl6jCO6z83vR9MaycaBhTMlNWL0
   ```

4. Если всё правильно, вы получите сообщение:
   ```
   ✅ Бот @ManikurAnna_bot подключён!
   Клиенты теперь могут записываться через: https://t.me/ManikurAnna_bot
   ```

---

## Готово!

Теперь у вас есть **свой бот** для записи клиентов. Отправьте клиентам ссылку:

```
https://t.me/ManikurAnna_bot
```

Клиент откроет вашего бота, нажмёт Start и сможет записаться к вам.

## Частые вопросы

### Где взять ссылку на моего бота?

Отправьте `/mybot` в @Clientikabot — бот покажет ссылку.

### Как отключить бота?

Отправьте `/removebot` в @Clientikabot.

### Куда приходят уведомления о записях?

В чат с @Clientikabot. Когда клиент запишется через вашего бота — вам придёт сообщение.

### Что будет если я потеряю токен?

Ничего страшного. Зайдите в @BotFather, отправьте `/mybots`, выберите вашего бота и нажмите «API Token» — увидите токен снова.

### Можно ли поменять имя или фото бота?

Да! В @BotFather:
- `/setname` — изменить имя
- `/setuserpic` — установить аватарку
- `/setdescription` — описание (видно при первом открытии)

### Бот не подключается, пишет ошибку

- Проверьте, что скопировали токен **полностью** (с двоеточием и буквами после него)
- Убедитесь, что между `/addbot` и токеном стоит **пробел**
- Если BotFather заблокировал бота — разблокируйте его командой `/revoke` и используйте новый токен
